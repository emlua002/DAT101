1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Constants should be used for values that needs to remain fixed throughout the program. This can for example be the value of PI, speed of light, types of limits, days in a week, etc. Variables should be used for values that can change throughout a program, for example counters, user inputs, results. No, not all values can be constants, considering there are at multiple occasions where values needs to change.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We can use the command "toFixed()" to limit floating-point numbers to a given number of decimals. However, this command returns a string, which means we have to convert it back to a number if we want that. Here is an example with both the fixed decimal command and the conversion back to a number::
    let number = 87.654321;
    let rounded = Number(number.toFixed(2));
    This will give us the rounded result of 87.65

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. In theory, we could use the "===" operator to compare to floating-point values to see if they are equal, since this is a comparison operator. But, since floating numbers are stored differently in a computer than whole numbers, this will not always work. Instead we could either round both numbers to a certain number of decimals, or we could check if the difference between these two numbers is smaller than a certain threshold. For example:
    let num1 = 12.199999;
    let num2 = 12.2;
    let threshold = 0.00001; //Defines a small threshold value that we consider "close enough"
    if (Math.abs(num1 - num2) < threshold) {
    console.log("The numbers are equal");
    } else {
    console.log("The numbers are not equal");
    }
    By making this function we can run these commands and get the correct result read back to us.

  6. How does the Math.random() function work, and what can it be used for?

    7. The Math.random() function generates a random floating-point number between 0 and 1 (this number can be 0, but never 1) This can be used to for example create random values for games, to get a random number for testing or to pick a random item from a list. If i want it to be a number between 0 and 100 for example, i can use the code:
    let randomNum = Math.random() * 101;
    This will give me a random floating-point number between 0 and 100. However, if i want it to be a whole numbers i can use the code:
    let randomWholeNum = Math.floor(Math.random() * 101);
    Math.floor will round the number down to the nearest whole number, if i want it to get rounded up instead, i would use Math.ceil

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. Modulus (%) is a mathematical operator, that returns the remainder of a division. Instead of dividing two numbers normally, for example 10 / 3 = 3.33333, we could write 10 % 3, which would give us the the remainder of the devision, which is one. Because 10 / 3 is 3 with a remainder of 1. We need this operator in mathematical operations when we need to check if a number is even or odd. We can check if a number is even by checking if the number % 2 is equal to 0.If it is, the number is even and if its not, its odd. Example of this:
    let number = 9;
    if (number % 2 === 0) {
    console.log("The number is even");
    } else {
    console.log("The number is odd");
    }
